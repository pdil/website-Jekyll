Description

PDColorPicker provides the ability to easily add a color selection interface to an iOS app. The provided view allows the user to select any 16-bit color by sliding across the different hue, saturation, and brightness values. The user interface's colors and fonts are easily customized to suit the appearance of any app in which PDColorPicker is used. It is available through CocoaPods, Carthage, and manual installation.

PDColorPicker is open for collaboration with the community so anyone is invited to submit issues or pull requests.
